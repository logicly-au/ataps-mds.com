Date

Valid values

  • YYYY-MM-DD (eg. 1980-12-31)
  • YYYY/MM/DD (eg. 1980/12/31)
  • DD-MM-YYYY (eg. 31-12-1980)
  • DD/MM/YYYY (eg. 31/12/1980)

The regular expression for validating dates is m/\d{4}[-\/]\d{2}[-\/]\d{2}/a

The date cannot be before the 1st of January, 1900.
