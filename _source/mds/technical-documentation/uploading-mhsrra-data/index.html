Uploading MHSRRA data

The ATAPS MDS has been upgraded to allow Mental Health Services in Rural and Remote Areas (MHSRRA) data capture.

This document outlines the changes to the full ATAPS upload file specification necessary in order to upload MHSRRA data.

Patients Upload File:

No Changes.

Referrals Upload File:

A referral type of 'MHSRRA' will now be allowed.

When a referral type of MHSRRA is specified the following referrer types are allowed:

  • GP
  • Psychiatrrist
  • Paediatrician (where the age is less than 15 years)

When a referral type of MHSRRA is specified the following extra fields are allowed:

  • F1 Alcohol & Drug Use
  • F2 Psychotic Disorders
  • F3 Depression
  • F4 Anxiety Disorders
  • F5 Unexplained Somatic

Sessions Upload File:

When a referral type of MHSRRA is specified the following session types are allowed:

  • Individual
  • Group
  • Family

When a referral type of MHSRRA is specificed the following session modalities are allowed:

  • Face to Face
  • Telephone
  • Video Conference
  • Web-based
