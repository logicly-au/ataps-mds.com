ATAPS Help - Mental Health Professional Data

The MDS provides an online form for MHPros professional data to be entered, as part of the ATAPS Evaluation.  Click on the 'Manage MHPros' link located on the MDS Home page.

You first need to locate the MHPro code in the MDS.  There are two ways of doing this:

1. 'List all" will list all existing MHPro codes in the MDS for your PHN. 

2. Alternatively, the "Search" function can be used to locate a specific MHPro code.

Once the MHPro code has been located, click on the 'Edit' link next to the appropriate record.

Now click on the "Create Details" button located on the top of the screen.

Complete the online form with the required details then click on the 'Save' button.
