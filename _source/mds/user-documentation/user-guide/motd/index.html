ATAPS Help - Message of the Day

The message of the day displays on the site's landing page when a user is not logged in.

The message is generated via an RSS feed from the announcements on the https://ataps-mds.com/ website.

In order to add or update the message, please login to the https://ataps-mds.com/ administrative interface and add or edit the relevent announcement.
